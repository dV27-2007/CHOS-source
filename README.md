# CH OS

CH OS is a custom minimal desktop layer on top of Ubuntu + i3wm.

Stack:
- Ubuntu
- i3wm
- kitty
- rofi
- ranger
- custom ch-os-status bar

Style:
- old school cyber terminal
- dark background
- green/turquoise colors
- no white/yellow UI

Main colors:
#030606
#071010
#06110f
#0f2f29
#137a66
#1edbb2
#2fffd0

Keybinds:
Alt+D       Application launcher
Alt+Esc     Power menu
Alt+Enter   Terminal
Alt+Tab     Next window

Tools:
chos-info          system info panel
chos-hotkeys       hotkeys panel
chos-maintenance   backup/check/restore panel
chos-control       control center
chos-check-i3      check i3 config
chos-save-stable   save stable point
chos-restore-i3    restore stable i3 config

Notes:
Alt+D uses rofi drun and shows installed apps automatically.
Terminal theme is intentionally not changed.
Old custom status bar is preserved.
